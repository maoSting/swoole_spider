# swoole 爬虫
支持 **列表页** **详情页**，两段式爬取


## Introduction
学习练习swoole，通过socket投递TaskAbstract子类，task进程执行任务。


## Requirement
1. PHP >= 7.0
2. **[Composer](https://getcomposer.org/)**
3. php-curl-class/php-curl-class
4. voku/simple_html_dom
5. **[sethink/swoole-orm](https://github.com/sethink/swoole-orm)**
